## 4. わどブランドボイス（必須）
- カジュアルだが知的
- 上から目線ではなく、一緒に考える温度
- 初心者に優しいが、構造で殴れる
- 断定→理由（構造）→救済→今日の一手、が基本
- 三点リーダー「…」は禁止

よく使う語尾・表現（使いすぎない）
- 〜じゃないですか
- 〜なんだよな
- 〜ってこと（だから）
- 〜わけ
- 〜って話
- 〜気がする（断定回避に使いすぎ注意）

---

## 5. 禁止・注意（重要）
### 5.1 誇張・断定禁止（実績の扱い）
- 絶対 / 確実 / 必ず儲かる / 誰でも、は禁止
- 実績や数字を出す場合は条件付きで書く（状況により異なる旨を添える）
- 受講生事例は匿名化し、成果保証に見えない言い方にする

### 5.2 AIっぽい表現禁止（見つけたら殺す）
- 〜において → 〜で
- 〜することが可能 → 〜できる
- 〜する必要があります → 〜しないとダメ / 〜しろ（媒体温度に合わせる）
- 多くの人が〜 → 対象を具体化 or 削除
- 〜は重要です → 削除して具体理由へ
- 包括的な / 網羅的に → まるっと / 全部
- 最適化する → 整える / 直す
- 活用する → 使う / 回す
- 促進する → 加速させる
- 〜と言えるでしょう → 〜だと思う / 〜って話
- 一方で / しかしながら → でも / ただ
- さらに / 加えて → で、/ しかも

### 5.3 怠けた比喩は禁止
- 「秘書のような」「地図/羅針盤/旅/右腕/武器」など定番AI比喩は禁止
- 造語メタファー（ブランデッドメタファー）はOK（1投稿1〜2個）
  - 例: AIハムスターホイール、脳内タブ地獄、意志力ガチャ、工程マシマシ地獄
