## 各パートの詳細

### 0:00-0:05 オープニング・自己紹介

- 自己紹介（簡潔に30秒）
- 「今日のゴール」を宣言:
  - 90分後に手元にある3つの成果物: SNS投稿10本 / プレゼンスライド / 自己紹介ページ
  - コードは一切書かない。コピペと日本語の指示だけ
- 参加者に「ClaudeCode触ったことある?」をチャットで確認

### 0:05-0:15 デモ3連発:「ClaudeCodeの衝撃」

最初に「何ができるか」を短いデモで見せてワクワクさせる。すべて実用的なデモ。

**デモ1（3分）: SNS投稿を一瞬で10本生成**
- 「AI活用をテーマに、X投稿を10本作って」
- 10本がずらっと出てくる → 「毎日の投稿で悩む時間がゼロになる」

**デモ2（4分）: プレゼンスライドをその場で作る**
- 「ClaudeCodeの紹介スライドを5枚、HTML形式で作って」
- ブラウザで開く → きれいなスライドが表示される
- 「提案資料・セミナー資料・勉強会用スライドが5分で作れる」

**デモ3（3分）: 自己紹介ページを作る**
- 名前・肩書き・得意なことを伝える → HTMLページが完成
- ブラウザで開いて見せる → 「これが10分で、無料で作れる」

「この3つを、今日の90分で皆さんにも全部やってもらいます。」

### 0:15-0:30 ハンズオン1: インストール＆セットアップ

- **Mac**: ターミナル → Homebrew確認 → Antigravityインストール
- **Windows**: PowerShell（管理者） → Wingetインストール
- 画面共有しながら1ステップずつ進行
- チャットで進捗確認:「できた人は 'OK' と打ってください」
- Anthropicアカウント作成 → `claude` コマンドで認証
- プロジェクトフォルダ作成
- よくあるエラー対処を都度画面で見せる

### 0:30-0:50 ハンズオン2: 自分のSNS投稿10本を作る（メインハンズオン）

ここが今日のメインイベント。参加者が**本当に使えるもの**を作る。

1. チャットにプロンプトテンプレートを貼る
2. 参加者に「テーマ」と「口調」を自分で決めてもらう（1分）
3. ClaudeCodeに入力して実行（2-3分）
4. 出力された10本を確認
5. 修正指示を体験:「3番目をもっとカジュアルに」→ 修正版が出る
6. **「これが対話型AI活用です」**
7. チャットで感想共有

ポイント: 遊びではなく**明日から使える成果物**を持ち帰る。

### 0:50-1:05 ハンズオン3: プレゼンスライドを作る

SNS投稿の次は「見せるもの」を作る。

1. チャットにテンプレートを貼る:
   ```
   以下のテーマでプレゼンスライドを5枚、HTML形式で作ってください。
   1枚ごとにセクション分けして、ブラウザで閲覧できるようにしてください。
   デザインはシンプルで見やすく。

   テーマ: ★自己紹介 / 自分の仕事紹介 / サービス紹介★
   スライド構成:
   - 1枚目: タイトル（名前・肩書き）
   - 2枚目: 自己紹介・経歴
   - 3枚目: 得意なこと・提供サービス
   - 4枚目: 実績・数字
   - 5枚目: お問い合わせ・SNS
   ```
2. ★部分を自分の情報に置き換えて実行（3-5分）
3. ブラウザで開いてスライドを確認
4. 「3枚目の背景を青にして」「フォントを大きくして」→ 修正体験
5. **「営業資料もセミナー資料も、これで作れます」**

### 1:05-1:15 ハンズオン4: 自己紹介ページを作る

4. 修正指示で微調整
