ClaudeCode導入ガイド
========================

このガイドを見ながら、ClaudeCodeを自分のPCで動かせる状態にしましょう。
プログラミング経験は不要です。コピペだけで完了します。

所要時間: 約20-30分


========================================
ステップ0: ClaudeCodeとは？
========================================

ClaudeCodeは「実行するAI」です。

普通のChatGPTやClaudeは「会話するAI」。
質問すると答えが返ってくるだけ。

ClaudeCodeは違います。
指示を出すと、AIが実際にファイルを作ったり、
Webページを生成したり、データを整理したりします。

例えば:
- 「SNS投稿を10本作って」 → すぐ使える投稿が出力される
- 「プレゼンスライドを5枚作って」 → HTMLスライドが完成する
- 「自己紹介ページを作って」 → おしゃれなWebページが出来上がる

このガイドでは、インストールから
実際に動かすところまでを説明します。


========================================
ステップ1: 事前準備
========================================

必要なもの:
- PC（Mac or Windows）※スマホ・タブレットでは使えません
- インターネット接続
- Anthropicアカウント（無料で作成可能）

料金について:
- ClaudeCode自体は無料でインストールできます
- AIを使うにはAnthropicのサブスクリプション（月額$20〜）が必要です
- 最初は無料枠で試すこともできます


========================================
ステップ2: Antigravityのインストール
========================================

Antigravityは、ClaudeCodeを動かすための土台となるソフトです。

---------- Macの場合 ----------

1. ターミナルを開く
   - キーボードで Command + スペース を押す
   - 「ターミナル」と入力してEnter
   - 黒い画面が出てきます（これがターミナルです）

2. 以下をコピーして、ターミナルに貼り付けてEnter

   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

   ※すでにHomebrewが入っている方はスキップしてOK
   ※「already installed」と出たらスキップ

3. 続けて以下をコピーして貼り付けてEnter

   brew install antigravity

4. パスワードを聞かれたら、Macのログインパスワードを入力
   ※画面に何も表示されませんが、入力されています。そのままEnter

---------- Windowsの場合 ----------

1. PowerShellを管理者権限で開く
   - スタートメニューで「PowerShell」と検索
   - 「管理者として実行」を選択

2. 以下をコピーして、PowerShellに貼り付けてEnter

   winget install antigravity

3. インストールが完了するまで待つ（1-2分）

---------- 確認方法（共通） ----------

以下を入力してEnter:

   antigravity --version

バージョン番号が表示されればOK！


========================================
ステップ3: ClaudeCodeのセットアップ
========================================

1. ターミナル（Mac）またはPowerShell（Windows）で以下を入力:

   claude

2. ブラウザが自動で開きます
   - Anthropicのログイン画面が表示されます
   - アカウントがない方は「Sign up」から作成
   - Googleアカウントでのログインが一番簡単です

3. ブラウザで認証が完了すると、ターミナルに戻ります
   - 「Welcome」的なメッセージが出ればOK

※ ブラウザが開かない場合:
   → まずブラウザでhttps://console.anthropic.com にアクセスしてログイン
   → その後、ターミナルで再度 claude を実行


========================================
ステップ4: 最初のプロジェクトを作る
========================================

1. 作業フォルダを作ります。以下を1行ずつ実行:

   mkdir my-first-project
   cd my-first-project
   claude

   - mkdir = フォルダを作るコマンド
   - cd = そのフォルダに移動するコマンド
   - claude = ClaudeCodeを起動するコマンド

2. ClaudeCodeが起動したら、チャット画面が表示されます

3. 権限の確認が出たら「Allow」「Yes」で許可してOK


========================================
ステップ5: SNS投稿10本を作ってみよう！
========================================

ClaudeCodeに以下を入力してEnter:

   以下の条件でX(Twitter)投稿を10本作ってください。
   - テーマ: AI活用のコツ
   - 口調: カジュアル
   - 1投稿140文字以内
   - 投稿の種類を混ぜてください（事例紹介/豆知識/Q&A形式/体験談/名言引用）

テーマは自分の発信内容に変えてOKです。
10本の投稿が表示されたら成功！

気に入らない投稿があれば「3番目をもっと短くして」のように
追加指示を出すだけで修正できます。


========================================
ステップ6: プレゼンスライドを作ってみよう！
========================================

ClaudeCodeに以下を入力してEnter（★を自分の情報に置き換え）:

   以下のテーマでプレゼンスライドを5枚、HTML形式で作ってください。
   1枚ごとにセクション分けして、ブラウザで閲覧できるようにしてください。
   矢印キーでスライドを切り替えられるようにしてください。

   テーマ: ★自己紹介★
   スライド構成:
   - 1枚目: タイトル（★名前・肩書き★）
   - 2枚目: 自己紹介・経歴
   - 3枚目: 得意なこと・提供サービス
   - 4枚目: 実績
   - 5枚目: お問い合わせ・SNS

完成したHTMLファイルをブラウザで開くと、スライドが表示されます。
「背景を青にして」「文字を大きくして」などの追加指示で修正できます。


========================================
よくあるエラーと対処法
========================================

エラー: 「command not found」
原因: Homebrew（Mac）またはWinget（Windows）がインストールされていない
対処: ステップ2の手順1からやり直す

エラー: 「Permission denied」
原因: 管理者権限が必要
対処: Mac → コマンドの前に sudo を付けて再実行
      Windows → PowerShellを「管理者として実行」で開き直す

エラー: 認証画面が出てこない
原因: ブラウザとの連携がうまくいっていない
対処: 先にブラウザで https://console.anthropic.com にログイン → 再度 claude を実行

エラー: 「No API key」「Authentication failed」
原因: サブスクリプションが未設定
対処: https://console.anthropic.com でMaxプラン（$20/月）を設定

何も表示されない（フリーズしたように見える）:
→ 処理中の可能性が高い。30秒〜1分待ってみてください。
→ 本当にフリーズした場合は Ctrl+C で中断して再実行。


========================================
Mac / Windows 操作の違いまとめ
========================================

ターミナルを開く:
  Mac: Command+スペース → 「ターミナル」
  Windows: スタートメニュー → 「PowerShell」（管理者として実行）

パッケージ管理:
  Mac: Homebrew（brew install ...）
  Windows: Winget（winget install ...）

パスワード入力:
  Mac: 画面に何も表示されない（正常です）
  Windows: 通常通り表示される

フォルダの場所:
  Mac: /Users/あなたの名前/
  Windows: C:\Users\あなたの名前\


========================================
--dangerously-skip-permissions について
========================================

ClaudeCodeの起動時に
「--dangerously-skip-permissions」
というオプションを見かけることがあります。

これは「毎回の権限確認をスキップする」設定です。

通常、ClaudeCodeはファイルを作成・編集するたびに
「これを実行してもいいですか？」と確認してきます。

このオプションを付けると確認なしで実行されます。
慣れてきたら使うと作業が速くなりますが、
最初は付けずに使うことをおすすめします。
