## ターゲット3層

1. **T1: 残業多め会社員副業トライアル層** — 30-45歳、年収400-600万、フック:「1日30分で月5万副収入」

**共通3大悩み**:
- AIを触っても"単発止まり"
- 時間がないのにやることが増える
- 「自動化」が怖い・難しい

**理想状態**:
1. AI秘書が「今日の材料」と「下書き」を用意している
2. 週10時間の余白が生まれ、副業の作業時間が確保できる
3. "時短x収益"の導線ができる（仕組みが資産になる）

## マーケティング

**Big Idea**: 「AIは難しい」→「Claude Code Desktopなら日本語指示だけでAI秘書が作れる。コード不要」

**訴求軸**:
- 成果:「3日間で動くAI秘書を作り、持ち帰れる」（T1, T3）
- 時間短縮:「1日30分のAI活用で月5万円の副収入」（T1）
- 不安解消:「AI時代に乗り遅れる前に」（T2）
- 損失回避:「AIエージェント開発で単価3倍」（T3）

**反論処理10個**:
1. プログラミング経験ゼロ → 日本語指示だけ、参加者60,000名超の大半が非エンジニア
2. 198,000円は高い → 買い切り、月々8,250円、1日275円、3名共有可
3. 忙しい → アーカイブあり、自分のペースで
4. 他社との違い → マンツーマン指導+買い切り+ツール提供型
5. 稼げるのか → 3日間で「動くもの」を作る体験が次の行動を生む
6. 全日参加できない → アーカイブ可、ただしFBはリアルタイムのみ
7. AI秘書って具体的に → メール下書き、議事録、SNS投稿、リサーチ等
8. 挫折が心配 → 90日間挫折しない仕組み（添削+生放送+コミュニティ）
9. 返金保証 → あり（一般申込）。わど経由は返金保証なし
10. ChatGPTで十分 → Claude Code Desktopはファイル操作・自動化・成果物生成に特化

**コンテンツ戦略**: 3コアメッセージ x 5投稿スタイル = 15パターン
**競合**: AI系スクール25社以上。差別化=ユニコ直接指導+買い切り+ツール提供型
