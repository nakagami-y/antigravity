## MCPとは何か

MCP（Model Context Protocol）は、Claude Codeを他のツール・サービスとつなぐ仕組みです。

たとえば:
- 「明日の予定を教えて」→ Google Calendarから予定を取得して回答
- 「先週の議事録を探して」→ Notionから該当ページを検索して表示
- 「タスクを作って」→ Asanaにタスクを自動作成

イメージとしては、AI秘書に「外部ツールを使う権限」を渡すようなものです。

## MCPでできること

| つなぐ先 | できること | 使用例 |
|---------|----------|--------|
| Google Calendar | 予定の確認・作成 | 「来週の空いている時間を教えて」 |
| Notion | ページの検索・作成 | 「〇〇のドキュメントを探して」 |
| Slack | メッセージの送受信 | 「チームに進捗を共有して」 |

| Asana | タスクの管理 | 「今週の未完了タスクを一覧にして」 |
| Gmail | メールの検索・下書き | 「山田さんへの返信を下書きして」 |

## 実践1: Google Calendar連携で「予定を聞くAI秘書」

### セットアップ手順（約10分）

**Step 1: MCP設定ファイルを作る**

Claude Code Desktopで以下を伝えます。

```
Google Calendarと連携するMCP設定を作ってください。
```

AIが `.mcp.json` という設定ファイルを作成します。

**Step 2: Google Cloud Consoleで認証設定**

1. https://console.cloud.google.com にアクセス
2. 新しいプロジェクトを作成
3. 「Calendar API」を有効化
4. 認証情報（OAuth 2.0）を作成
5. ダウンロードした認証ファイルを指定の場所に保存

**Step 3: 初回認証**

```
Googleカレンダーの予定を確認してください
```

初回はブラウザが開いてGoogleアカウントでの認証を求められます。「許可」を押せば完了です。

### 使い方

認証が完了したら、自然な日本語で予定を聞けます。

```
「明日の予定を教えて」
「来週の月曜日に14時から1時間の打ち合わせを入れて。タイトルは企画会議」
「今週の空いている時間帯をリストアップして」
「3月の予定を一覧にして」
```

## 実践2: Notion連携で「ナレッジ検索AI秘書」

### セットアップ手順（約10分）

**Step 1: Notion APIキーを取得**

1. https://www.notion.so/my-integrations にアクセス
2. 「新しいインテグレーション」をクリック
3. 名前をつけて作成
4. 表示されるAPIキーをコピー

**Step 2: Notionのページに権限を付与**

検索対象にしたいNotionページを開き、右上の「...」→「コネクト」→ 作成したインテグレーション名を選択

**Step 3: MCP設定に追加**

```
Notion MCPを設定してください。APIキーは [コピーしたキー] です。
```

### 使い方

```
「先月の会議メモを探して」
「プロジェクトAの進捗をNotionから確認して」
「新しいページを作って。タイトルは "3月の振り返り"」
```

## 実践3: Slack/Discord連携

### Slack連携の概要

Slack連携では以下ができます:
- 特定チャンネルのメッセージを検索
- メッセージの送信
- チャンネル内の会話を要約

**セットアップの流れ:**
1. Slack Appを作成（https://api.slack.com/apps）
2. 必要な権限（スコープ）を設定
3. ワークスペースにインストール
4. Bot Tokenを取得
5. MCP設定ファイルに追加

### Discord連携の概要

Discord連携では以下ができます:
- サーバー内のメッセージを同期
- 特定チャンネルの内容をローカルファイルに保存
- メッセージの検索

## MCP連携の全体像

```
あなた ←→ Claude Code Desktop ←→ MCP ←→ 外部ツール
                                    |
                         ┌──────────┼──────────┐
                         |          |          |
                    Calendar     Notion     Slack
                    (予定管理)  (ナレッジ)  (コミュニケーション)
```

複数のツールを同時に連携できます。たとえば:

```
「明日の打ち合わせ相手のNotionページを探して、
 議題を箇条書きにまとめて、
 Slackで相手に送って」
```

このように、複数ツールをまたいだ操作を1回の指示で実行できます。

## トラブルシューティング

### 「認証エラー」が出る

**原因**: APIキーの期限切れ、または権限不足
**対処**:
1. 各サービスの管理画面でAPIキーを再発行
2. `.mcp.json`のキーを更新
3. Claude Code Desktopを再起動

### 「ツールが見つからない」と言われる

**原因**: MCP設定ファイルの記述ミス
**対処**:
1. `.mcp.json`の内容を確認
2. JSON形式が正しいかチェック（カンマ、括弧の閉じ忘れ）
3. 設定ファイルのパスが正しいか確認

### 「応答が遅い」

**原因**: 外部APIの応答待ち
**対処**:
1. インターネット接続を確認
2. 対象サービスがダウンしていないか確認
3. タイムアウト設定を調整（通常は30秒で十分）

### 「権限がない」と言われる

**原因**: 連携先サービスでの権限設定不足
**対処**:
- Google: Cloud ConsoleでAPIを有効化しているか確認
- Notion: ページにインテグレーションの権限を付与しているか確認
- Slack: Bot Tokenのスコープに必要な権限が含まれているか確認
