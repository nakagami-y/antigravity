# Claude Code 即効プロンプト30選 — コピペするだけで今日から使える

**プログラミング知識ゼロでもOK！**

---

## この資料の使い方

たった3ステップで使えます。

**ステップ1:** 使いたいプロンプトをコピーする
**ステップ2:** Claude Codeに貼り付ける
**ステップ3:** 実行する（Enterキーを押すだけ）

プロンプトの中に `{　}` で囲まれた箇所があります。そこだけを自分の情報に書き換えてください。たとえば `{取引先名}` なら「株式会社〇〇」に、`{商品名}` なら「マッサージオイル」に変えるだけです。

難しいことは何もありません。まずは1つ、気になるプロンプトをそのまま使ってみてください。

## ビジネス編（プロンプト1〜10）

### プロンプト1 — 営業メールの下書き

取引先への提案メールを5分で仕上げたいときに。

```
以下の条件で、取引先への提案メールを書いてください。

宛先: {取引先名} {担当者名}様
自分の会社名: {自社名}
提案内容: {提案する商品・サービス名}
提案の目的: {アポイント獲得 / 資料送付の許可 / 商談の設定 など}
相手との関係: {初めての連絡 / 以前名刺交換済み / 既存顧客}
メールのトーン: ビジネス丁寧語（重くなりすぎず、読みやすい長さで）

件名も一緒に考えてください。
```

ポイント: 「相手との関係」を書くだけで、初対面と既存顧客で文体が自動的に変わります。
