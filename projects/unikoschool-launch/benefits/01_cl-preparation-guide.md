## 事前準備（所要時間: 約15分）

### 1. Claude Code Desktop をインストールする

Claude Code DesktopはAnthropicの公式AIツールです。以下の手順で準備してください。

**必要なもの:**
- パソコン（Mac または Windows）
- インターネット接続
- Anthropicアカウント（無料登録可）

**手順:**
1. https://claude.ai にアクセスしてアカウントを作成
2. Claude Code Desktop をダウンロード
3. インストールしてログイン
4. 画面左下に「Claude Code」と表示されていれば完了

**料金について:**
- Proプラン（月額$20）がおすすめ
- 3日間チャレンジの間だけ使って、不要なら解約してもOK
- 課金開始前に無料枠で動作確認できます

### 2. 作業フォルダを作成する

デスクトップに「ai-secretary」というフォルダを作ってください。Day 1でこのフォルダを使います。

```
デスクトップ/
  ai-secretary/     ← このフォルダを作成
```

### 3. やりたいことを1つ決めておく

AI秘書に最初にやらせたいことを1つ考えてきてください。

- 議事録の要約
- メールの下書き
- SNS投稿の作成
- レポートの整理
- スケジュール管理

何でもOKです。「こんなことできるかな？」くらいの軽い気持ちで大丈夫です。

## よくある質問

**Q: プログラミング経験ゼロでも大丈夫？**
A: 大丈夫です。Claude Code Desktopは日本語で指示するだけ。参加者の大半が非エンジニアです。

**Q: 3日間全部参加できないかもしれない**
A: アーカイブ視聴が可能です。ただし完走特典は全日参加+課題提出が条件です。

**Q: パソコンはMacじゃないとダメ？**
A: WindowsでもMacでも使えます。スマホからの参加は推奨しません。

**Q: 子どもがいて途中退出するかも**
A: 途中入退出は自由です。アーカイブで復習できます。

**Q: 事前にClaude Codeを触ったことがないけど大丈夫？**
A: Day 1で基本操作から一緒にやるので、事前経験は不要です。上の「事前準備」だけ済ませてきてください。
