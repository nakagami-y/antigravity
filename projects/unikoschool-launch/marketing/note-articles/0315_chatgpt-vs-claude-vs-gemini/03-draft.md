「結局、どのAIを使えばいいの?」

AIを使い始めようとした人が、最初にぶつかる疑問です。ChatGPT、Claude Code、Gemini。それぞれ名前は聞いたことがある。でも何が違うのか、どれが自分に向いているのかが、わからない。

この記事では、3つのAIを「会話力」「仕事力」「創作力」の3軸でまとめます。

最後に、Claude Codeの始め方ガイドを無料で配っています。記事を読んだあとにLINEを登録するだけで受け取れます。

## まず、3つのAIの「立ち位置」を整理する

それぞれのAIには、得意な領域があります。ひとことで言うとこうです。

- **ChatGPT** → 会話する。何でも答えてくれる汎用型
- **Gemini** → 見る・作る。画像生成とGoogle連携が強い
- **Claude Code** → 動かす。日本語で指示するだけで仕事を自動実行する

この3つは「競合」ではなく、用途が違うツールです。ただ、副業や仕事の効率化を目的にするなら、選ぶべきAIは明確に変わります。

---

## ChatGPT — 「話し相手」として最強

ChatGPTの強さは、会話の自然さと汎用性にあります。

「この文章を要約して」「アイデアを出して」「英語で言い換えて」。何を聞いても、それなりの答えが返ってきます。試しに使い始めるなら、ChatGPTが一番とっつきやすい。

ただ、ChatGPTは「答えを出すAI」です。

あなたが質問を送る。ChatGPTが答えを返す。それで終わりです。ファイルを開いたり、定刻に動いたり、複数の作業を連続でこなしたりすることは、基本的にできません。

副業で使おうとしたとき「毎日30分かけて指示を打ち込んでいる」という状況になりやすいのはそのためです。ChatGPTはあくまで会話ツールなので、自分で指示し続ける手間は変わりません。

副業や仕事に活かしたいなら、ChatGPTは「考えるパートナー」として使うのが現実的です。

---

## Gemini — 「画像と検索」に強い

GeminiはGoogleが開発したAIです。他の2つと比べて、画像・動画・音声への対応が厚いのが特徴です。

たとえばこういう使い方が得意です。

- アイデアスケッチを描いたら、それをもとに画像を生成してもらう
- YouTubeの動画URLを貼り付けて、内容を要約してもらう
- GmailやGoogleドキュメントと連携して、メールの下書きをそのまま送る

GmailやGoogleドキュメントをすでに日常的に使っているなら、Geminiとの連携はかなり便利です。Googleサービスの中にGeminiが入り込んでいるので、切り替えなしで使えます。

一方で、「決まった時間に動く」「複数の作業を連続で実行する」は、GeminiもChatGPT同様に弱い部分です。Geminiは「リッチな会話型AI」という位置づけが近いです。

---

## Claude Code — 「仕事をさせる」AIはこれだけ

ここが、この記事の核心です。

ChatGPTとGeminiは「話すと答えが返ってくる」AIです。Claude Codeは違います。「日本語で指示すると、AIが実際に手を動かして仕事を完成させる」AIです。

具体的に何ができるか、いくつか実例を挙げます。

**SNS投稿カレンダーを15分で作る**
「3月のAI活用テーマで、月曜事例・水曜豆知識・金曜Q&A形式のSNS投稿を1ヶ月分作って、CSVで出力して」と伝えると、ファイルが完成します。30日分の投稿案を毎回ゼロから考える時間がなくなります。

**月次レポートを10分で終わらせる**
売上のCSVファイルを渡して「前月比・上位5商品・達成率をまとめてExcelで出して」と言うだけ。1時間かかっていた集計作業が自動で終わります。

**セミナー告知ページを20分で作る**
「イベント名・日程・参加費・申込フォームURL・こんな雰囲気のデザインで」と伝えると、HTMLのWebページが出てきます。そのままアップロードすれば公開できます。

**毎日のルーティン作業をワンコマンド化する**
「Skills」という機能を使うと、「/sns-calendar」「/monthly-report」のように短いコマンドを打つだけで、毎回同じ品質の成果物が出てくる状態を作れます。

---

この差が出る理由は、Claude Codeが「実行するAI」だからです。

ChatGPTやGeminiは「会話の中で答えを作る」ツールです。Claude Codeはファイルを作り、計算し、HTMLを書き、データを変換する。会話ではなく、仕事を完成させます。

さらに重要なのが、コードを一行も書く必要がないことです。「英語が読めない」「プログラミングが全然わからない」でも、日本語で指示するだけで動きます。

---

ただ、一点だけ確認してほしいことがあります。

Claude Codeには「Desktop版（Claude Code Desktop）」と「CLI版」の2種類があります。CLI版はターミナルという黒い画面を使うもので、エンジニア向けです。

非エンジニアが使うのは**Claude Code Desktop**の方です。ブラウザで開いて、日本語で話しかけるだけで動きます。「コードが書けないから難しいかも」という不安は不要です。

## 3軸で比べると、こうなる

3つのAIを「会話力」「仕事力」「創作力」で整理します。

**会話力（質問に答える・アイデアを出す）**
- ChatGPT → 最強。汎用性の高さと自然さは群を抜く
- Gemini → 得意。Google検索との連携で情報精度が高い
- Claude Code → 十分。ただし会話よりも実行に強みがある

**仕事力（ファイル作成・データ処理・自動実行）**
- ChatGPT → 限定的。基本は会話止まり
- Gemini → 一部対応。Googleサービス連携に限る
- Claude Code → 圧倒的。ファイル生成から自動実行まで日本語だけで動く

**創作力（画像生成・デザイン・ビジュアル）**
- ChatGPT → DALL-E連携で対応。品質は安定している
- Gemini → 最強。Imagen連携で高品質な画像生成が得意
- Claude Code → テキスト・コードが中心。画像生成は専門ではない

---

用途で選ぶとシンプルです。

「文章を書くのを手伝ってほしい」「アイデアを一緒に考えたい」→ ChatGPT

「画像を作りたい」「Gmail・Googleドキュメントと連携したい」→ Gemini

「ファイルを自動で作りたい」「毎回やっている作業を自動化したい」「副業に活かしたい」→ Claude Code

## 「AIと話す」時代から、「AIに仕事させる」時代へ

2年前までのAIの使い方は、「質問して答えをもらう」でした。ChatGPTが登場したとき、多くの人が感動したのはそこです。

いま、その次のステップが来ています。

「AIに話しかけるのではなく、AIに仕事を渡す」という使い方です。

ChatGPTで下書きを作ってもらっていた人が、Claude Codeに切り替えたら「毎月2時間かかっていたレポートが10分になった」という声があります。Excelの集計、SNSの投稿案、告知ページの作成、議事録の整理。毎回ゼロから手を動かしていた作業が、一度設定すれば自動で動くようになります。

副業をやりたいけど時間がない。本業が忙しくて手が回らない。子育ての合間に収入の柱を作りたい。

そういった状況で「時間を作る手段」として、Claude Codeが機能します。

AIは道具です。道具は使い方で価値が変わります。「会話するだけ」から「仕事を任せる」に切り替えると、使える時間の量が変わります。
