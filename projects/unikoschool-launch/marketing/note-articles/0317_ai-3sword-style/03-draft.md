# NotebookLMで勉強、Geminiで下書き、Claude Codeで仕上げ。AI3刀流の1日を公開

「AIを使いたいけど、何をどう使えばいいかわからない。」

ツールが多すぎる。ChatGPT、Gemini、Claude、Copilot、NotebookLM。どれを使っていいかわからないまま、結局ChatGPTに質問するだけで終わっている。

その状態を抜け出す方法が、今日の話だ。

答えはシンプルだ。3つだけ使う。朝・昼・夜で役割を分ける。それだけで1日の仕事が変わる。

最後に、この3刀流を3日間で体験できるイベントの話もする。

---

■ 3刀流の全体像

まず、使うツールを決める。

- 朝: **NotebookLM**（Googleの音声要約AI）
- 昼: **Gemini**（Googleの画像・文章生成AI）
- 夜: **Claude Code**（Anthropicの自動実行AI）

3つ選んだ理由がある。

この3つは「役割が被らない」からだ。NotebookLMはインプット専用。Geminiは素材作り専用。Claude Codeは完成物の自動生成専用。それぞれが違う仕事をしている。

朝は情報を頭に入れる。昼は素材を作る。夜はアウトプットを出す。

この流れに沿って動けば、頭で考える量が減る。

---

■ 朝: NotebookLMで「ながら勉強」する

▸ NotebookLMとは

GoogleのAIツールで、PDFや記事のURLを読み込んで「対話形式の音声」に変えてくれる。

ポッドキャストのような形式で、2人のAIが内容をかみ砕いて話してくれる。日本語対応済みだ。

▸ 実際の使い方

毎朝やっていることがある。

前日に気になった記事や、読み終えていないPDFをNotebookLMに入れておく。朝の支度をしながら、その音声を流す。

料理をしながら、洗濯物をたたみながら、子どもの送り出しをしながら。

手が塞がっていても、情報が入ってくる。

- PDFを入れる → 音声化 → 家事しながら聞く
- ビジネス書のまとめを入れる → 通勤中に聞く
- 仕事の参考資料を入れる → 10分で概要をつかむ

「読む時間がない」という問題が変わる。

読む → 聞く、に変わるだけで、インプットの量が倍になった感覚がある。

▸ T2（ママ層）への効果

育児をしていると、まとまって座れる時間が少ない。

NotebookLMは「座らなくていい勉強」を作ってくれる。家事をしながら、子どもと遊びながら、頭だけ動かせる。

朝の30分が「ながら学習の時間」に変わる。

---

■ 昼: Geminiで画像とデザインの下書きを作る

▸ Geminiとは

GoogleのAI。文章を書くだけでなく、画像を生成できる。テキストから画像を作ったり、Google WorkspaceのスライドやGmailと連携したりする。

▸ 実際の使い方

SNS投稿用のバナー画像を作るとする。

デザインソフトが得意でない人が、Canvaで1時間かけて作っていたものが、Geminiなら5分で下書きが出る。

```
正方形サイズ（1:1）で作ってほしい。
背景はシンプルな白。
テキスト: 「3つのAIで仕事が変わる」
サブテキスト: 「今日から始める時短術」
配色はダークネイビーとオレンジで、シンプルに。
ビジネス系のSNS投稿用。
```

→ 数秒でデザイン案が出てくる。細かい修正は追加指示でできる。

資料の図解を作るときも同じだ。

```
「作業量が減る → 余白が増える → 思考が深まる」
この流れを図解で表してほしい。
矢印でつながった横並びの3つのボックス形式で。
```

→ 図解の下書きが出てくる。

「作る」作業から「確認して調整する」作業に変わる。これが昼のGeminiの役割だ。

▸ ポイント

Geminiはインプットに「何を作りたいか」をできるだけ具体的に書くと精度が上がる。

- サイズ（1:1、16:9など）
- 使用用途（SNS投稿、資料、説明図）
- テイスト（シンプル、カラフル、プロフェッショナル）
- テキスト内容

この4つを入れるだけで、修正回数が激減する。

■ 夜: Claude Codeで成果物を「自動生成」する

ここからが、3刀流の本番だ。

朝と昼は「自分がツールを使う」話だった。夜のClaude Codeは「AIが勝手にやる」話になる。これが質的に違う。

**▸ 朝・昼との違い**

- NotebookLM: 自分が情報をインプットする
- Gemini: 自分が素材を作る
- Claude Code: AIが成果物を自動で仕上げる

「使う」から「任せる」への転換が、夜のClaude Codeで起きる。

**▸ Claude Codeとは**

Anthropicのツール。ブラウザで動く。インストール不要。コード不要。英語も不要。

日本語で「何をしてほしいか」を書くだけで、ファイルを操作したり、文章を生成したり、複数の作業を一括でやったりする。

ChatGPTとの違いを一言で言うと、ChatGPTは「答えを教えてくれる」がClaude Codeは「実際にやってくれる」だ。

**▸ 夜のルーティンに入れた使い方**

1日の終わりに、その日あったことをメモで渡す。

```
今日の仕事の内容:
- 取引先Aとの打ち合わせ（来月の施策について議論）
- SNS投稿を3本作成（エンゲージメント目的）
- 社内向けのレポートを作成（月次まとめ）

これをもとに:
1. 明日のTODOリスト（優先順位付き）を作って
2. 打ち合わせの要点を3行でまとめて
3. 明日使えるSNS投稿案を2本作って
```

→ 3つの成果物が一度に出てくる。

「明日の準備」が10分で終わる。

**▸ 毎週やっていること**

週末に1回、SNS投稿を一括生成している。

```
来週1週間分のSNS投稿（X向け）を作って。
テーマ: AI活用・時短・働き方
口調: 親しみやすい、知識感がある
1投稿140字以内
各投稿に改行を入れて読みやすくして
合計5本
```

→ 5本が一覧で出てくる。

1週間分を30分で作る。毎日投稿内容を考える必要がなくなる。

■ 「スキマ時間の質」が変わる

3刀流を使い始めて変わったことがある。

「作業する時間」が減って「確認する時間」に変わった。

- NotebookLM: 読む → 聞く（手が塞がっていても動く）
- Gemini: 作る → 調整する（ゼロから作らなくていい）
- Claude Code: 考える → 確認する（AIが案を出して、自分は選ぶだけ）

1つひとつは小さい変化だ。でも3つ合わせると、1日の「手を動かしている時間」がかなり減る。

仮に毎日30分ずつ削減できれば、週で2.5時間。月で10時間。

その余白で何をするかは、自分で決められる。子どもと過ごす時間かもしれない。副業に使う時間かもしれない。休む時間かもしれない。

使い方の話ではなく、時間の使い方の話だ。

1日の終わりに「今日の内容をまとめて、明日の準備をして」と言うだけで動く。
