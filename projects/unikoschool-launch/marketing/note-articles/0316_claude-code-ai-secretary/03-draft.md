# Claude Codeで「AI秘書」を作ってみた。コード0行で何ができたか全部見せる

「AIに仕事を任せたい。でもプログラミングは無理。」

そう思ったまま、ChatGPTとの"おしゃべり"で終わってる人に伝えたい。

Claude Codeという選択肢がある。

ターミナルを開かなくていい。コードを書かなくていい。英語も要らない。

日本語で「毎朝の業務レポートを自動で作って」と入力するだけで、本当に動くものが5分で完成する。

この記事ではその全工程を、プロンプトから完成物まで全部公開する。

最後に、これを3日間で体験できる無料イベントの話もする。

---

## ChatGPTとClaude Code、何が違うのか

最初にここだけ整理しておく。

ChatGPTは優秀な**会話相手**だ。質問すれば答えてくれる。相談すればアドバイスをくれる。でも仕事を「実行」はしない。

Claude Codeは**仕事仲間**だ。

「このフォルダにあるメモを整理して」と言えば整理する。「毎朝このデータをまとめてレポートを作って」と言えばそのレポートを作る。「あの取引先への返信メールを下書きして」と言えばメールの文面が出てくる。

ChatGPTが「どうすればいいか教えてくれる人」なら、Claude Codeは「実際にやってくれる人」だ。

そしてClaude Code Desktopは、ブラウザで動く。

## AI秘書を作る。全工程見せる

### Step 1: Claude Code Desktopを開く

ブラウザからアクセスするだけでいい。

アカウント登録は必要だが、設定作業はほぼゼロ。インストール不要。ターミナルも要らない。

画面を開くと、チャット画面が出てくる。ここに日本語で指示を打ち込む。それだけだ。

### Step 2:「毎朝の業務レポートを自動で作って」と指示する

実際に入力したプロンプトがこれだ。

```
私は毎朝、以下の作業を手作業でやっています。
・昨日完了したタスクを箇条書きにする
・今日やること3つを優先順位付きで並べる
・懸念事項があれば1行で記録する

このレポートを毎朝自動で作れるように、
テンプレートと入力フォームを作ってください。
私が「昨日の完了タスク」「今日の予定」「懸念事項」を入力するだけで、
整形されたレポートが一瞬で出るようにしてほしい。
```

プログラミングの知識はゼロでいい。「こういうものを作ってほしい」を日本語で書くだけでいい。

### Step 3: 5分で完成物が出てくる

Claude Codeはこう返してきた。

```
了解しました。毎朝のレポート作成を自動化します。

以下のフォームに入力するだけで、整形されたレポートが出力されます。

---
【業務日報テンプレート】

日付: [自動入力]

▸ 昨日の完了タスク
1.
2.
3.

▸ 今日のTOP3（優先順位順）
1. 【最優先】
2.
3.

▸ 懸念事項
・ なし or 内容を1行で

---

このテンプレートを使う場合、毎朝このチャットに以下の形式で送ってください。

「昨日: 〇〇完了、△△完了
今日: ××の資料作成、□□への連絡、◇◇確認
懸念: なし」

そうすれば即座に整形してレポートにします。
```

5分どころか、1分で返ってきた。

しかもこれは「提案」ではなく「実際に動くもの」だ。翌朝から使える状態で出てきた。

## AI秘書にできること、全部見せる

上で作ったのは日報テンプレートだけだが、同じ要領でできることがある。実際に試した5つを紹介する。

**▸ メール下書き**

```
〇〇株式会社の田中さんから、先週の打ち合わせのフォローアップメールが来ました。
会議では来週までに見積もりを出す話をしていました。
ただ今週は社内確認が間に合わなさそうで、1週間延期をお願いしたいです。
丁寧なビジネスメールで返信の下書きを作ってください。
```

→ 宛名、お詫びの文、延期理由、次のステップまで整った文面が出てくる。

修正するのは固有の数字や日付だけだ。

**▸ 議事録整理**

会議中に走り書きしたメモをそのままペーストして送る。

```
（乱雑なメモ）
A社 来月から月次定例 毎月第2火 10時
予算は承認待ち 太郎さんが確認する
次回に担当者紹介してもらう
懸念: 担当変更の可能性
```

→ 「議題 / 決定事項 / Next Action / 懸念事項」の形式に整理して返してくれる。

**▸ SNS投稿案**

```
対象: 30代の忙しい会社員
口調: 親しみやすい、でも知識感がある
テーマ: 朝のルーティンをAIで効率化する話

この設定でX（Twitter）の投稿案を10本作って。
各投稿は140字以内、改行を入れて読みやすくして。
```

→ 10本の投稿案が一覧で出てくる。そのまま使えるものが7〜8本は含まれている。

**▸ リサーチレポート**

```
最近話題のAI議事録ツールについて調べたい。
主要な3〜4つのサービスを比較して、
「機能 / 料金 / 使いやすさ」を整理したレポートを作って。
対象: AI初心者の総務担当者が読む前提で。
```

→ 箇条書きで整理されたレポートが出てくる。自分でGoogle検索する時間が要らなくなる。

**▸ 優先順位付きTODO整理**

```
今日やることが山積みで頭が整理できていません。
以下のリストを優先順位順に並べ替えて、
所要時間の目安と「最初にやるべきこと」を1行で教えて。

・来週の資料を作る
・Aさんへの返信
・経費精算（締め切り今日）
・定例会議の準備
・Bプロジェクトの進捗確認
```

→ 締め切り・緊急度・重要度を考慮して並べ替えてくれる。「最初にやること: 経費精算（今日締め切り・所要15分）」と一行でまとめてくれる。

## 「コード0行」の意味を正確に伝える

ここで一度、整理する。

Claude Code Desktopを使うのに、以下は**一切不要**だ。

- ターミナル（黒い画面）を開くこと
- Pythonや何かのプログラミング言語
- 設定ファイルを書くこと
- 英語でコマンドを打つこと

必要なのはただひとつ。**日本語で「何をしてほしいか」を書くこと**だけだ。

「でも、具体的にどう書けばいいかわからない」という人のために言うと、今この記事に書いたプロンプトをそのままコピーして使える。変えるのは固有名詞だけでいい。

それだけで動く。

## なぜ今、「AI秘書」なのか

2024年からAIは急速に使える道具になった。

でも多くの人は「ChatGPTに質問する」で止まっている。

AIとの会話で終わっていて、仕事の自動化にはつながっていない。

その理由のひとつは「プログラミングが要るのでは」という思い込みだ。

Claude Code Desktopはその壁を壊す。コードゼロで、仕事を動かす。メールを下書きする。資料を整える。レポートを生成する。

こういった**実行型AIの使い方**を身につけた人と、ChatGPTとのおしゃべりで止まっている人とでは、1年後の仕事量が変わってくる。

仕事の量ではなく、仕事に使う時間が変わる。
