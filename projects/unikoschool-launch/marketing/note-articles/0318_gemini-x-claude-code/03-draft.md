# Geminiの画像生成が神。Claude Codeと組み合わせたら副業の生産性が3倍になった話

Geminiが画像を生成できるようになった。

しかもそこらのAI画像ツールとはレベルが違う。テキストを入力するだけで、プロのデザイナーが作ったような画像が数秒で出てくる。SNSのバナー、note記事のサムネイル、プレゼンの図解。全部タダで、全部30秒以内に。

「これで副業のデザイン費が浮く」と思った。

実際、浮いた。

でも問題が起きた。**画像は作れる。でもそれだけでは仕事にならない。**

バナーを作ったあと、記事を書いて、SNS投稿文を用意して、メルマガに組み込む。この「その後」がぜんぶ手作業だった。

画像生成で節約できた時間を、後工程で全部使っていた。

その問題をClaude Codeが解決した。

最後に、この「Gemini × Claude Code」の組み合わせを3日間で体験できるイベントの話もする。

---

## Geminiの画像生成は何がそんなにすごいのか

まず事実だけ整理する。

Googleが2025年末にリリースした**Gemini 2.0 Flash**は、テキストから画像を生成できる。しかも無料で使える範囲が広い。

何がすごいかというと、**日本語の指示がそのまま通る**ことだ。

英語プロンプトを覚える必要がない。「Canvaの使い方を勉強する」必要もない。「note記事のサムネイル。白背景、タイトルは『AIで副業を10倍にした話』、右下にシンプルな図解を入れて」と日本語で打てば、それが出てくる。

実際に試した結果を並べる。

**試したケース1: SNSバナー**

- 指示: 「副業AI活用セミナーのTwitterバナー。紺背景、白文字。タイトルは『AIで1日2時間を生み出す方法』。右側にシンプルなスマホのイラスト」
- 結果: 30秒で出てきた。Canvaで作ったら30分かかる仕上がりだった

**試したケース2: note記事サムネイル**

- 指示: 「白背景。中央に大きく『ChatGPTとClaude Codeの違い』。下に3つの比較ポイントを箇条書きで図解」
- 結果: 読みやすいインフォグラフィックが完成。自分でPowerPointを触る必要がなかった

**試したケース3: プレゼン用図解**

- 指示: 「Before→Afterの矢印図。左は手作業の仕事フロー（4時間）、右はAI活用後のフロー（30分）。シンプルなアイコン使用」
- 結果: 伝えたい内容がそのまま図になった

この3つ、全部無料だ。タダで、30秒以内に。

Canvaの有料プランを解約した人を何人か知っている。

## 「でも画像だけでは稼げない」という壁

ここで現実の話をする。

Geminiで画像を作れるようになった。すごい。感動した。でも副業の収益は変わらなかった。

理由は単純だ。**コンテンツビジネスは「画像1枚」では成立しないから**。

副業でnote記事を書いている人を例に挙げる。

1本の記事を公開するまでに何が必要か、洗い出してみた。

- サムネイル画像: 1枚
- 記事本文: 3,000〜5,000字
- 記事内の図解: 3〜5枚
- X（Twitter）への告知投稿: 3〜5本
- Instagramのリール用キャプション: 1本
- メルマガ本文: 1本

画像を素早く作れるようになっても、その他が全部残っている。

「Geminiで画像を作って、残りを手作業でやる」状態が続いた。

1本のコンテンツを仕上げるのに、画像生成20分 + 残り作業4時間という構造は変わらなかった。

## Claude Codeを組み合わせたら全部変わった

ここからが本題だ。

Claude Codeに「記事を書いて、画像の指示も出して、SNS投稿も作って」とまとめて依頼した。

最初はどうせ雑なものが出てくると思っていた。

出てこなかった。

実際に使ったプロンプトを公開する。

---

```
テーマ: Geminiの画像生成を使った副業効率化

このテーマで以下を全部作って。

① note記事本文（3,000字程度）
  - ターゲット: 副業でコンテンツを作っている30代
  - 構成: 問題提起→Gemini活用法→限界→Claude Codeとの組み合わせ→まとめ
  - 語尾はですます調、短文を意識して

② 記事内の図解の指示（3枚分）
  - 各図解に「何を表現する図か」と「Geminiへの画像生成指示文」をセットで

③ X（Twitter）投稿文（5本）
  - 各140字以内
  - 読んでクリックしたくなる書き出し
  - ハッシュタグは各投稿2〜3個

④ Instagramキャプション（1本）
  - 500字以内
  - 読者が保存したくなる内容

⑤ メルマガ本文（800字程度）
  - 件名も含めて
  - 読んで記事に飛びたくなるテキスト
```

---

5分後に全部出てきた。

**Before（以前の作業時間）:**
- 記事本文: 3時間
- 図解の準備（Canva作業）: 1時間
- SNS投稿文: 30分
- メルマガ: 30分
- 合計: **5時間**

**After（Claude Code使用後）:**
- 上記プロンプトを打つ: 3分
- Claude Codeの出力確認・微調整: 20分
- 図解をGeminiで実際に生成: 15分
- 合計: **38分**

5時間が38分になった。

「3倍」と言ったが、実際には8倍近い。保守的に言っても、同じ時間で3〜4本の記事を出せるようになった。

副業でコンテンツを出す頻度が変わると、収入の安定性が変わる。1週間に1本から3本に増えた人がいる。

## 「本当に自分でもできるか」という疑問に答える

ここで一番多い反応を先に言っておく。

「プログラミングが必要なんでしょ？」

要らない。

Claude Codeは日本語で動く。今この記事に書いてあるプロンプトをそのままコピーして使える。変えるのはテーマ名と自分のターゲット読者像だけだ。

「Geminiの使い方がわからない」

Googleのアカウントがあれば始められる。チャット画面に日本語で「〇〇の画像を作って」と打つだけだ。英語プロンプトは不要。

「どこから始めればいい？」

Geminiで画像を1枚作ってみることから始める。今日の記事に使うサムネイルでいい。「白背景、タイトルは〇〇、シンプルな図解入り」と打ってみる。

それだけでいい。難しいことは何もない。

## 副業でコンテンツを出し続けるための構造

最後に、少し先の話をする。

コンテンツビジネスで最も難しいのは「質」よりも「継続」だ。

質は後から上がる。でも継続できない人が多い。理由のひとつは、1本作るのに時間がかかりすぎて疲弊するからだ。

週1本出せれば年52本。週3本出せれば年156本。

この差は1〜2年後に大きく出る。

「Gemini × Claude Code」の組み合わせは、このボトルネックを解消する。

時間がかかっていた「コンテンツの量産」が、時間のかからない作業になる。あとは自分の経験とエピソードを乗せるだけだ。

そこにしか他との差はつけられない。

道具を使いこなした人が、継続できる。
