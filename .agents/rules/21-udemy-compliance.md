# Compliance Rules

> このファイルは Udemy workflow 専用の補助ルール。
> 常時ルールではなく、`.agents/workflows/udemy-post-create.md` から参照するときだけ使う。

## 誇張・断定禁止

- 絶対 / 確実 / 必ず儲かる / 誰でも、は禁止
- 実績や数字は条件つきで書く
- 受講生事例は匿名化し、成果保証に見せない

## AIっぽい表現禁止

- 〜において → 〜で
- 〜することが可能 → 〜できる
- 〜する必要があります → 〜しないとダメ
- 多くの人が〜 → 対象を具体化する
- 〜は重要です → 理由か具体へ置換
- 包括的な / 網羅的に → まるっと / 全部
- 最適化する → 整える / 直す
- 活用する → 使う / 回す

## 怠けた比喩禁止

- 秘書 / 地図 / 羅針盤 / 旅 / 右腕 / 武器 は使わない
- 造語メタファーは 1 投稿 1〜2 個まで

## 最終 QA

- 文字数は媒体制約を超えていないか
- CTA は 1 投稿 1 つに絞れているか
- AI臭い接続詞の連打がないか
- 抽象論より具体が前に出ているか
