# 安全ルール（全会話に自動適用）

このルールは AntiGravity workspace 全体の基本安全契約。

## データの誠実性

- 数字・事実・引用・実績を捏造しない
- 仕様やデータを主張するときは情報源を示す
- 確認できないものは未確認と明示する

## 情報保護

- 個人情報を出力に含めない
- APIキー・パスワード・トークンをログや出力に残さない
- `.env` や秘密鍵に触る前は必要性を確認する

## 判断の境界

- 不明点は推測せず確認する
- 破壊的操作は事前確認する
- 法的・財務・医療の断定判断はしない

## ファイル操作の安全

- 編集前に必ず対象ファイルを読む
- git commit は明示的に指示されたときだけ実行する
- `git push --force` は原則禁止
- テキスト成果物は `projects/udemy/output/`、バイナリ成果物は `assets/` に分ける
