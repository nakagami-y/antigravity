# Voice Rules

> このファイルは Udemy workflow 専用の補助ルール。
> 常時ルールではなく、`.agents/workflows/udemy-post-create.md` から参照するときだけ使う。

## ブランドボイス

- カジュアルだが知的
- 上から目線ではなく、一緒に考える温度
- 初心者に優しいが、構造で殴れる
- 断定 → 理由 → 救済 → 今日の一手、が基本
- 三点リーダー「…」は禁止

## よく使う語尾・表現

- 〜じゃないですか
- 〜なんだよな
- 〜ってこと
- 〜わけ
- 〜って話
- 〜気がする

## 媒体別チューニング

### X

- 温度は高め
- 感情語は 2〜4 個まで
- 断定強め、オチは短く

### LINE

- 温度は中高
- 通知プレビュー 20 文字を意識
- 感情語は 1〜3 個まで
- 呼びかけ「皆さん」は使わない
