# Strategy Workflow

## 目的

- 方向性の妥当性を検討し、次の一手まで落とす

## 手順

1. 人間からの指示に対する plan を root `memory_bank/plan/` に作る
2. 目的との接続を確認する
3. いまの前提が有効かを見る
4. 選択肢を並べる
5. 得るものと捨てるものを書く
6. 最小の検証アクションを 1 つ決める
7. root `memory_bank/report/` に report を作る

## 出力

- Recommendation
- Reasoning
- Trade-offs
- Next Step
- Confidence
