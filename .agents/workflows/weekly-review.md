# Weekly Review Workflow

## 目的

- 週次の振り返りを、次週の 1 アクションまで整理する

## 手順

1. 人間からの指示に対する plan を root `memory_bank/plan/` に作る
2. 前回記録があれば確認する
3. 今週の highlight を 1 つ出す
4. challenge を 1 つ出す
5. 学びを 2 つ以内に絞る
6. 次週の action を 1 つに固定する
7. root `memory_bank/report/` に review を残す

## 保存先

- root `memory_bank/report/`
