# 株式会社UnionAI — 組織体制

## 組織構成の概要

株式会社UnionAIは、役員3名＋稼働メンバー11名の計14名体制で運営されている（2026年2月時点）。B2C事業とB2B事業を2名のGM（ゼネラルマネージャー）が分担し、CEOが全社統括を行う構造である。

| 項目 | 内容 |
|------|------|
| **役割** | 全社経営の最終責任者 |

**責任範囲:**

## 6カテゴリ工数配分

全社の工数を6カテゴリに分類し、戦略に応じて配分を管理する。

### カテゴリ定義

| カテゴリ | 定義 | KPI指標 |
|---------|------|---------|
| **集客** | 認知→LINE登録を増やす | LINE新規登録数 / リーチ / フォロワー増 |
| **ファネル** | LINE→セミナー→成約の仕組み | CVR / セミナー参加者数 / 配信到達率 |
| **商品力** | 売れる商品を作り続ける | 教材完成度 / サブスク基盤 / NPS |
| **CS** | 既存顧客の満足→継続→紹介 | 解約率 / 応答速度 / 紹介数 |
| **Sales** | 面談→成約 | 成約数 / 成約率 |
| **社内** | 経理・法務・総務・採用 | 締め遵守率 / ミス件数 |
