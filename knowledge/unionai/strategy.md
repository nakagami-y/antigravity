## 最大ボトルネック: LINE→面談CVR

### 現状のファネル数値（2026年1月実績）

```
リーチ（X + YouTube + note + IG + オプチャ等）
  ↓
LINE登録 428名/月
  ↓ CVR 0.3%（★最大のボトルネック）
面談予約 約1.3件/月（CVRベース）
  ※ 実際の面談51件はセミナー経由・DM経由等を含む
  ↓ 成約率 12%
成約 6件 → 月商510万円
```

### ボトルネック分析

| ファネルステップ | 現状 | 目標 | ギャップ倍率 | 優先度 |
|----------------|------|------|:-----------:|:------:|
| **LINE→面談予約CVR** | **0.3%** | **3.0%** | **10倍** | **最優先** |
| 成約率 | 12% | 25% | 2.1倍 | 高 |
| LINE新規登録数 | 428名 | 3,000名 | 7倍 | 中（Phase 2以降） |
| 面談キャパシティ | 51件 | 60件 | 1.2倍 | 低 |

### なぜファネルCVRが最大の制約か

1. **集客を増やしてもCVRが0.3%のままだと漏れるだけ** — LINE 3,000名集めてもCVR 0.3%なら面談9件。目標60件に届かない
2. **CVRを先に上げれば、少ない集客でも面談が増える** — CVR 3.0%になれば現状の428名でも面談13件。集客増で掛け算が効く
3. **Sales改善もCVR改善の後** — 面談数が足りなければ成約率を上げても絶対数は増えない

### Phase別ボトルネック解消の優先順位

| Phase | 最大の制約 | 対策 | 解消後の状態 |
|-------|----------|------|------------|
| Phase 1 | ファネルCVR 0.3% | 配信シーケンス整備・セミナー定期化・特典見直し・LP改善 | CVR 1.0%で面談40件確保 |
| Phase 1 | 成約率 12% | トーク標準化・事前教育強化 | 25%で月10件成約 |
| Phase 2 | 集客数 428名 | YouTube本格化・note SEO・IG強化 | 1,200名に拡大 |
| Phase 3 | オートウェビナー未稼働 | AutoBwinner導入 | 24h稼働で面談枠拡大 |
| Phase 4 | 広告なしの集客上限 | 広告テスト開始 | 3,000名/月達成 |

### マスターコンテンツ戦略

コンテンツは「ネタ収集 → 素材蓄積 → 大マスター制作 → チャネル展開 → 反応回収 → ネタに戻る」のループで運用する。

#### 3ティア構造

| ティア | 定義 | 文字数 | 例 |
|--------|------|--------|-----|
| 大マスター | 深い構造の長文コンテンツ | 5,000字以上 | note記事 / YouTube台本 / セミナー台本 |
| 中マスター | 教育・ナーチャリング用 | 1,000-3,000字 | メルマガ / Q&A / 事例記事 |
| 素材 | 未加工のインプット | 短文〜メモ | 作業ログ / メモ / 気づき / 面談ノート |

#### 展開フロー

1つの大マスターから以下を展開する:
- X投稿 x 3-5本
- ショート動画台本 x 1-2本
- メルマガ x 1通
- LINE配信 x 1通
- オプチャ投稿 x 6OC
- IG投稿 x 1-2本
- 中マスター（Q&A/事例）x 1本

## リスク管理

| コンテンツ制作が追いつかない | AI生成ナレーション活用、短尺5分動画で制作コスト削減 |
| 広告費ゼロの限界 | Phase 2以降で広告テスト（月10-30万円）を開始 |

## 方針メモ

- チャレンジローンチ（2025年12月）は目標2.25億円に対し実績約5,000万円（約22%）。目標のリアリティと段階的成長を重視する方針に転換
- 固定費・人件費の最適化、成約率・CVRの改善が重点
- 現状の月間人件費は約750万円で人件費率147%（赤字）。売上拡大と固定費管理の両輪が必要
