# わどのオファー体系

## 設計思想

すべての商品・サービスは「AI活用」を前提とする。

- AIで商品数を増やす
- AIで収益機会を増やす
- AIでサポートを自動化する

この3本柱で設計されていない商品は存在しない。商品は「売り込み」ではなく「AI診断結果に基づく処方箋」として提案する。

```
【0 --> 10 レーン】初心者層向け
  無料テンプレ（LINE配布）

```

## 10 --> 100 レーンの商品: コンテンツエージェント

初回ローンチを最短で回せる状態にし、次回以降も量産できる形（OS化）まで持っていくサービス群。

### コンテンツエージェントの目的

- 「何を売るか（オファー）」が競合基準で整理される
- 売るために必要な素材が一式で揃う、または量産できる
- 投稿や文章を"毎回ひねり出す"のではなく、型 x AIで増やせるようになる

### 共通の成果物（ローンチ一式）

どのコースでも最終的に揃える成果物は以下の5つ。

| # | 成果物 | 備考 |
|---|--------|------|
| 1 | LP（ランディングページ） | 商品の価値を伝え、申込みに導くページ |
| 2 | SNS投稿（1日1本ベース） | コンセプトに沿った導線投稿 |
| 3 | セミナー（スライド + 台本） | 体験 --> 納得 --> 個別相談への接続 |
| 4 | 個別相談（トークスクリプト） | ヒアリング --> 提案 --> クロージング |
| 5 | LINEステップメール | 教育 --> 誘導 --> 予約 --> 追客 |

画像生成は手順提供のみ（実制作・画像データの納品はしない）。

### SNS「1日1本」の定義

日記投稿ではなく、コンセプト（誰に / 何を / 何で勝つか）に沿った導線投稿。

- 悩みの言語化（ターゲットが"自分ごと化"する）
- 解決策の提示（わどの勝ち筋）
- 実績・事例・証拠
- よくある反論つぶし
- セミナー/LINE誘導（CTA）

## Audit（診断）の設計

### 目的

「正解の実物」と「現在地の数字」を渡し、ギャップを確定する。学習ではなく監査（Audit）--> ギャップを埋める手段として商品を提案する。

### Audit Kit（特典として配布）

| 特典 | 内容 |
|------|------|
| 実案件のブループリント | 導線全貌図、分岐条件、KPI基準値 |
| 機会損失チェッカー | 入力: リスト数/成約率/単価 --> 損失額を可視化 |

### 診断フォーム（足切り + 病状分類）

応募者全員にZoomを提供するのではなく、フォームで足切りと分類を行う。

| 条件 | ルーティング |
|------|------------|
| 月売上 0〜5万 | 0 --> 10 レーンへ自動誘導（個別相談不可） |
| 予算なし・独学 | 対象外（資料配布で終了） |
| それ以外 | Zoom（戦略セッション）へ |

### Audit --> 商品提案の流れ

```
コンテンツ（X/note/LINE/セミナー等）
  --> 問題の自覚が生まれる
    --> LINEで「診断」と送る
      --> 診断フォーム回答
        --> 足切り + 分類
          --> Zoom（戦略セッション）
            --> ギャップの可視化
              --> 処方箋としてAI x 商品提案
```

## 商品提案の原則

1. 商品は「AI x 処方箋」として出す（売り込みではなく、AI診断結果に基づく提案）
2. 「AIでこうすれば解決できる」という具体的な活用イメージを必ず添える
3. 価格はコンテンツ内に記載しない（個別相談/セミナー/販売ページでのみ提示）
4. 条件を満たさない人には上位商品を提案しない
5. すべての提案にAI活用3本柱（商品数UP / 収益機会UP / サポート自動化）のいずれかの視点を含める

## ヒアリングでの「隠れたスキル発掘」

多くの人は「自分にはスキルがない」と思い込んでいるが、セールス時のヒアリングで深掘りすると、十分なスキルを持っていることが多い。

### 発掘すべきスキルの例

- デザイン歴3年以上（趣味レベルでもOK）
- 話すのが得意（セミナー講師、コーチングの素質）
- 恋愛経験が豊富（恋愛系コンテンツに転用可能）
- 特定分野の深い知識（料理、育児、健康、趣味など）
- 「非稼ぐ系スキル」こそ掘り起こす価値がある

### ターゲット判定の基準

| 条件 | 分類 |
|------|------|
| 月10万以上稼いでいる | 10 --> 100 層 |
| 月5,000円以上 + 十分なスキルがある | 10 --> 100 層 |
| スキルはあるが稼げていない（デザイン歴3年等） | 10 --> 100 層 |
| 本当に何もスキルがない | 0 --> 10 層（少数） |

> 「ほとんどの人は何かしら努力している。全く何もない人はなかなかいない」

## 成約率を上げるポイント

1. **「具体的な次の一歩」を提示する** -- 具体的なアクションが見えると人は動ける。動けると成約する
2. **相手の状況に合わせてオファーを変える** -- 一律セールスをしない
3. **0 --> 10 層にはコンテンツ販売を推さない** -- イメージが湧かないから。まず「運用代行」「画像制作」等の具体的なアクションを提示
4. **中上級者には「労働集約型からの脱却」で訴求する** -- 痛みに直接刺す
