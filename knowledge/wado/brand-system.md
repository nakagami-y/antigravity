# わどブランドシステム

## 北極星（約束）

**『努力』ではなく『AIと仕組み』で勝たせる。**

AIで積み上がる資産（導線・教育・自動化）を高速構築し、スキルワーカーを「時間とお金の余裕がある状態」へ最短で導く。

| 要素 | 内容 |
|------|------|
| 約束 | 努力量ではなくAIと仕組みで勝たせる |
| 主戦場 | 10 --> 100（投稿の70%） |
| 一言定義 | 月10万で止まるスキルワーカーを、AI x 商品化 x 導線 x 自動化で伸ばす |
| 天才性 | AI x 効率構造のクリエイター -- 一撃で複数成果を生む |
| 対人スタンス | AI時代の知識の共有者 -- 囲わず出し、周りごと強くする |

## AI活用3本柱（全活動の前提）

わどのすべてのコンテンツ・商品・サービス・投稿はAI活用を前提とする。「仕組み化」と言ったら「AI x 仕組み化」を意味する。AIなしの仕組み化は提案しない。

### 柱1: AIで商品数を増やす

コンテンツ・商品・LP・メール・オファーの制作をAIで加速する。1人でも多品種展開し、ローンチ回数を増やす。

- AIでLP/コンテンツを高速で作れる
- 1つの商品を起点に派生商品を量産できる
- 制作のボトルネックがなくなり、打席数が増える

### 柱2: AIで収益機会を増やす

AIでローンチを量産し、テスト --> 改善 --> 再ローンチの高速サイクルを回す。打席数 = 売上。

- ローンチ準備期間が従来の1/3に短縮
- A/Bテストをスピーディに回せる
- 1回の大勝負ではなく、AIで何度もチャレンジできる

### 柱3: AIでサポートを自動化する

AIエージェントで顧客対応・教育・フォローを24時間稼働。人手ゼロで回る仕組みを構築する。

- 受講生の質問対応をAIが担当
- 教育コンテンツの配信を自動化
- 人的リソースに依存しないスケーラブルな運営

## 2レーン定義

わどのコンテンツ・商品・導線はすべて2つのレーンに分かれている。1つのコンテンツには1つのレーンのみを割り当てる（混在禁止）。

### レーンA: 0 --> 10（未マネタイズ〜月10万未満）

| 項目 | 内容 |
|------|------|
| ラベル | 0 --> 10 |
| 対象 | まだ売上がない、または月10万円未満のスキルワーカー |
| ゴール | AIを使って初売上・初レビュー（最小の成功体験）を最速で達成 |
| 投稿の着地 | AIでローンチ素材（LP/動画/メール/オファー）を「作れる」ようにする |
| AI活用の訴求 | AIでコンテンツ制作を加速し、商品数・ローンチ回数を増やして打席に立つ |
| 商品の役割 | 成功体験 --> 壁を体感 --> 低単価商品 --> 上位商品へ |
| CTA | 無料テンプレ（LINE配布で統一） |

#### 0 --> 10 の人に刺さるテーマ

- AIを使って最初のローンチ素材を作る方法（LP、メール、LINE）
- スキルゼロでも大丈夫? 「隠れたスキル」を見つける方法
- コンテンツ販売の前にまずやるべき「小さな一歩」
- 10万円で始められるAIビジネスブートキャンプとは

### レーンB: 10 --> 100（主戦場）

| 項目 | 内容 |
|------|------|
| ラベル | 10 --> 100 |
| 対象 | スキル収益月10万程度 OR フォロワー5,000以上（商品未完成OK） |
| ゴール | AIを使って集客と販売の「仕組みが自動で回っている状態」 |
| 投稿の着地 | AIでローンチ回数を増やし、数字（CV/申込/成約/LTV）を高速で改善できるようにする |
| AI活用の訴求 | AIで商品数を増やす / AIでローンチを量産する / AIエージェントでサポートを自動化する |
| オファーの核 | 学習ではなく「監査（Audit）」--> ギャップを埋める手段としてAI x 商品提案 |
| CTA | 応募制「事業ボトルネック特定診断（Audit）」--> Zoom --> 本命商品 |

#### 10 --> 100 の人に刺さるテーマ

- ローンチ回数をAIで3倍に増やす方法
- AIでCV率を上げる具体的テクニック
- 労働集約型ビジネスから脱却するためのコンテンツ販売戦略
- 1回のローンチ準備を1/3の時間で終わらせる方法
- コンセプトを入力するだけでLP・メール・SNS投稿が出てくる仕組み

## 4レバー（10 --> 100 の投稿カテゴリ）

10 --> 100 レーンの投稿は、以下の4つのカテゴリに分類する。上から強い順。

### 1. 売上設計（最も強いレバー）

AIで商品数を増やす。単価UP・LTV最大化・オファー量産・ローンチ回数の増加。

- 「商品が1つしかない」= 弾切れビジネス。AIで商品を増やせ
- 単価を上げるには「学習」ではなく「環境構築」を売る
- ローンチを年1回から年4回にするだけで売上が変わる

### 2. 集客設計

AIでコンテンツを量産しリスト獲得を加速。広告・導線・SEOの最適化もAIで。

- マスターコンテンツ1本から10+のチャネル展開
- AIでフックを設計し、スクロールを止める
- リスト獲得を仕組み化して、毎月安定して集客

### 3. 自動化設計

AIエージェントでサポート・教育・フォローを自動化。人手ゼロで回る仕組み。

- 受講生対応を24時間AIで回す
- LINEステップメールの教育 --> 誘導 --> 予約 --> 追客を自動化
- 「人力ヘルプデスク地獄」から脱出する

### 4. 診断・添削

AI診断でボトルネック特定 --> LINE接続（最強のCTA接続点）。

- 機会損失チェッカーで損失額を可視化
- 「何が足りないか」をAIで特定し、次の一手を提示
- 診断結果 --> 個別相談（Zoom）--> 商品提案の自然な流れ

## CTA制限ルール

### 基本ルール

- **1コンテンツにつきCTAは最大2つ**
- レーン混在禁止: 0 --> 10のCTAと10 --> 100のCTAを同一コンテンツに混ぜない
- CTA先は固定（以下参照）

### レーン別CTA

| レーン | CTA先 | 導線 |
|--------|-------|------|
| 0 --> 10 | LINE --> 無料テンプレ配布 | コンテンツで学び --> LINEでテンプレ入手 --> 実践 |
| 10 --> 100 | LINE --> 応募制Audit診断 --> Zoom --> 処方箋（商品） | コンテンツで問題を自覚 --> 診断 --> 個別相談 --> 商品 |

### CTA表現例

#### 0 --> 10 向け

- 「AIで使えるこの型、LINEで無料配布してます」
- 「AI活用度チェックリストで今の状態を確認してみて」
- 「AIで商品を作る手順テンプレ、LINEで配布中」

#### 10 --> 100 向け

- 「AIで自分のボトルネックを特定したい人は、LINEで"診断"って送ってみて」
- 「月10万超えてるけど伸び悩んでる人は、AIで導線診断してます」
- 「AI活用の個別診断、応募制でN名だけ受付中」

## レーンラベル必須ルール

- 全コンテンツの冒頭またはメタデータにレーンを明示する: 「0 --> 10」 or 「10 --> 100」
- 1コンテンツ = 1レーン（橋渡しテーマでも主レーンを1つ決める）
- レーンが不明なコンテンツは制作しない

## 10 --> 100 の反論（不安）TOP6

投稿で先回りして潰すべき不安:

| # | 不安 | 回答の方向性 |
|---|------|-------------|
| 1 | 値上げしたら失注しそう | AIで商品価値を上げれば値上げは自然。価値 > 価格の状態を作る |
| 2 | 実績が弱い | AIで小さなローンチを量産し、実績を積み上げる。最初は数で勝負 |

## 禁止事項

### 誇大表現の禁止

- 「絶対」「確実」「必ず儲かる」「誰でも」などの断定は禁止
- 実績主張は具体的な条件付き: 「OOの場合にOOの結果が出た」
- 「月収OO万」系の主張は、条件・期間・母数を併記

### 恐怖訴求の禁止

- 「知らなきゃオワコン」「やらないと終わる」「取り残される」は使わない
- 煽りではなく構造的な問題提起 --> 仕組みによる解決の流れにする

### 他社否定の禁止

- 競合スクール・サービスの名指し批判は禁止
- 「他では」「普通のスクールは」などの暗示的否定も避ける

### 価格非表示

- 具体的な価格（OO万円）はコンテンツ内に記載しない
- 価格は個別相談/セミナー/販売ページでのみ提示

### 絵文字制限

- X投稿: 原則テキストのみ。使う場合は1投稿に最大2個
- note記事: 見出しのみ許可。本文は原則なし
- LINE/セミナー: 適度に使用OK（ただし1メッセージ3個以内）

---

## 実績主張ルール

- 自分の実績: 事実ベースで条件付き記載OK
- 受講生の実績: 本人許可があるもののみ。匿名化する場合は「受講生Aさん」等
- 捏造・誇張は絶対禁止
- 数字には出典を紐づける。確認できないものは「未確認」と明示する
