# 00_knowledge — 3ブランド マスター知識ベース

## このフォルダの目的

株式会社UnionAIが展開する3ブランドの**上流マスター定義**を格納する。
ブランドの「何者か」「誰に何を届けるか」「どう語るか」をここで定義し、下流の制作・運用スキルが参照する。

---

## 上流（定義層）と下流（運用層）の関係

| 層 | パス | 役割 | 更新頻度 |
|----|------|------|---------|
| **上流（定義）** | `00_knowledge/` | ブランドの本質・戦略・声・商品体系の定義 | 四半期〜半期 |
| **下流（運用）** | `01_projects/02_b2c-marketing/knowledge/` | 日常の制作で使うナレッジ（フック設計、CTA辞書等） | 週次〜月次 |

- 上流が変わったら下流に伝播する（ルール変更時の影響スキル特定 → `rules/70-skill-lifecycle.md`）
- 下流で発見した改善は、必要に応じて上流にフィードバックする
- **正（SSOT）はこのフォルダ**。下流は運用に特化した派生物

## 3ブランドの関係

```
┌─────────────────────────────────────────────┐
│  株式会社UnionAI（法人）                       │
│  unionai/                                    │
│                                              │
│  ┌──────────────────┐  ┌──────────────────┐  │
│  │  わど（個人Brand）  │  │  ひかり（YouTube）│  │
│  │  wado/            │  │  hikari/         │  │
│  └──────────────────┘  └──────────────────┘  │
└─────────────────────────────────────────────┘
```

| 項目 | わど | ひかり | UnionAI（法人） |
|------|------|--------|----------------|
| **ターゲット** | 10→100（中上級者） | 0→10（AI未活用のビジネス職） | 法人（研修・サブスク・受託） |
| **主チャネル** | X / note / LINE / セミナー | YouTube / note / X | 法人営業 / ひかりYT経由 |
| **人格** | わど本人 | ひかり（演者） | 法人としてのUnionAI |
| **集客の流れ** | X・note → LINE → セミナー → 成約 | YouTube → LINE → セミナー → 成約 | ひかり/わど経由 or 直接法人営業 |

### ブランド間の連携ポイント

- **集客**: わど（X/note）とひかり（YouTube）が異なるチャネルで異なる層を集め、LINE に合流する
- **商品**: B2C商品はわどスクールが中心。ひかりは集客エンジン。法人はあいらぼ+研修
- **コンテンツ**: わどのマスターコンテンツがひかりのYouTube台本の素材にもなる。逆にひかりの動画反応がわどのネタになる
- **人格**: わど = 中上級者向けの知的な語り口。ひかり = 初学者に寄り添う親しみやすさ。同じ事業でもトーンが異なる
