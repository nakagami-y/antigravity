# Note Writer Skill

箇条書きのメモをもとに、note向けの共感を呼ぶ記事を作成します。

## トリガー

- `/note-writer`
- 「note書いて」「記事にして」

## 入力

- 記事の元ネタとなる箇条書きメモ（必須）
- ターゲット読者（任意）

## ワークフロー

1. **構成の設計**: 提供されたメモをもとに、導入・本文・まとめの構成を立てる
2. **執筆**: noteらしい「共感」「有益性」を意識して本文を執筆する
3. **保存と表示**: 完成した記事をMarkdownファイルに保存し、画面にも表示する

## 出力

- **画面出力**: 記事の全文を表示する
- **ファイル保存**: `note_drafts/YYYY-MM-DD_タイトル.md` に保存する
  - フォルダがない場合は作成する

- [ ] 冒頭（リード文）で読者の関心を惹きつけているか？
